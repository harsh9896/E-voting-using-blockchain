User-Manual

There are few guidelines for the user:
  1. Voter Registration
    • For casting the vote user need to register himself.The registration form is provide in the left side of the system.click on it
    • The voter can only register in the registration phase.Once registration phase is over a voter cannot register.
    • For registration, user need to enter his aadhar number and his account address
    • At first stage the user age is verified only voter above 18 are allowed to register
    • Otp bad me team se discuss krke dekh lenge
  2. Voting Process

      voting process is divided into 3 phases:

    • Registration phase:Here the registration of user is carried out.
    • Voting Phase: After initialization of voting phase from the admin,user can cast vote.The vote can be cast by simple cicking the "voting" button,after which a transaction will take place from the user account to the selected candidate account. After voting phase, voter will not be able to cast vote
    • Result Phase: This is the final stage where the user can see the result who won the election.
